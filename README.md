Um E-commerce OpenSource voltado para o mercado Brasileiro.

## Instalação

Adicione esta linha ao fim do Gemfile da aplicação:

    gem 'escambo'

Depois execute:

    $ bundle

Ou instale manualmente:

    $ gem install escambo

Não esqueça de rodar as migrations: `rake escambo:install:migrations`
